# MotifVM Public Demo

Run all canonical MotifVM demos:

```bash
./demo/run_all.sh
```

Outputs:

- `demo/audit_packs/`
- `.motifvm/demo_outputs/benchmark.csv`
- `.motifvm/adversarial_outputs/adversarial_results.csv`

Open `demo/graph_explorer/index.html` in a browser and load any exported `state.json` from an audit pack.
